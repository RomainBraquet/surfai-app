## Décisions techniques
- Colonne Supabase = `board_type`, JS = `type` → mapping en lecture/écriture
- UUID Supabase nécessite des quotes dans les onclick HTML
- Suppression board = détacher sessions (SET board_id = NULL) puis DELETE
- Overlay partage dans `.app` container (420px), pas en fixed plein écran
- loadBoardsTab fetch toujours depuis Supabase (bypass cache)
- Géoloc accueil : réutilise _userLat/_userLng si connus, sinon demande permission
